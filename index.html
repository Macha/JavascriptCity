JavascriptCity by Macha

A javascript clone of Simcity

This is a personal project of mine to write a javascript clone of Simcity. In it's current city, it has about as much features as Simcity 0.5. It has zones, power and road. The biggest problem encountered while creating this was javascript performance as this is far beyond what the designers of javascript had in mind. The other problem was the actual table used to draw the game. When I started, I hadn't heard of , so the game uses a 40,000 cell table (200 x 200) as the game's grid.

Dependencies

PHP (optional)

Install

Short (needs PHP capable webserver): Place in a PHP capable webserver. Long: Remove the PHP code from the top of index.php Paste the contents of grid.php over the PHP code in the middle
